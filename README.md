# phejno-plugins

Personal Claude Code plugin marketplace.

## Installation

```bash
claude plugins marketplace add przemyslawhejno/phejno-plugins
```

## Available Plugins

### claude-project-index

Generates and maintains a compact project index in `.claude/index/` for fast file lookup. Reduces token usage by 60-80% on navigation tasks.

```bash
claude plugins install claude-project-index@phejno-plugins
```
